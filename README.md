**PantryBuddy** is a Java-based desktop application that serves as an interactive recipe database, designed to simplify meal planning and help users make the most of the ingredients they already have.

### Technical Overview
- **Language:** Java  
- **UI:** Swing with FlatLaf for a modern, responsive look  
- **Database:** SQLite (via JDBC) for local data storage  
- **External API:** TheMealDB (used for initial recipe data)  
- **Testing:** JUnit for unit and component testing  
- **Data handling:** Jackson for JSON parsing and persistence  

### Project Goals
- Improve everyday meal planning and cooking efficiency  
- Reduce unnecessary grocery shopping by reusing available ingredients  
- Provide a modular and extensible architecture for future features  
- Serve as a solid foundation for potential premium features (e.g. advanced filters, meal plans)

PantryBuddy is a stable, offline-capable, and user-friendly application that combines practical functionality with a clean and intuitive interface.
